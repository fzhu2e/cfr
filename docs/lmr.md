# LMR

`cfr` provides convenient workflows for a typical paleoclimate data assimilation (PDA) framework, the last millennium reanalysis (LMR, [Hakim et al., 2016](https://doi.org/10.1002/2016JD024751); [Tardif et al., 2019](https://doi.org/10.5194/cp-15-1251-2019)).

In this section, we will cover the below topics:

- [Real PAGES2k Reconstruction](notebooks/lmr-real-pages2k.ipynb)
- [Independent Verification](notebooks/lmr-real-pages2k-indpdt-verif.ipynb)
- [Assimilation without Calibration](notebooks/lmr-assim-no-calib.ipynb)
- [Reconstructing Precipitation](notebooks/lmr-recon-pr.ipynb)
- [CLI Usage](notebooks/lmr-cli.ipynb)
