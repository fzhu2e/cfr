# Proxy

`cfr` provides a toolkit for the analysis/visualization of the commonly used proxy records, including but not limited to the PAGES 2k global multiproxy database ([PAGES2k Consortium, 2017](https://doi.org/10.1038/sdata.2017.88)).

In this section, we will cover the below topics:

- [Proxy IO](notebooks/proxy-io.ipynb)
- [Proxy Operations](notebooks/proxy-ops.ipynb)
- [Proxy Analysis](notebooks/proxy-analysis.ipynb)
- [Proxy Composites](notebooks/proxy-composites.ipynb)
- [Proxy Duplicates](notebooks/proxy-dups.ipynb)
- [Loading DOD2k](notebooks/proxy-load-dod2k.ipynb)
